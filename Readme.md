# btl_he_dieu_hanh
Lệnh chạy project

- B1: cài tất cả package đang có trong project
```
sudo apt install
```

- B2: 
```
make all
```

- B3: How to Run the Simulation, chạy các file hiện có trong folder input, ví dụ file sched_0
```
./os sched_0
```

- để xóa file trong obj
```
make clean
```
